Bridge project

As a mechanical engineering student at Bahrain Polytechnic University, I was required to build a small scaled bridge that withstands a high load. This project helped improving my problem solving skills and it focused a lot on self learning and researching how can the bridge to withstand the highest load possible while only using 150 sticks to build the whole bridge which created a bigger challenge to complete it. The bridge was successfully designed and built after many trials and tweaks and it withstood a load of 96 kilograms.
